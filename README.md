# McGill Music Storefront

This folder contains a simple direct-sales storefront for beat checkout using PayPal.

## Files
- `index.html`
- `styles.css`
- `app.js`

## To activate PayPal checkout
1. Get your PayPal client ID from the PayPal developer dashboard.
2. Open `app.js`.
3. Replace `REPLACE_WITH_PAYPAL_CLIENT_ID` with the real client ID.
4. Adjust product names and prices as needed.

## Current assumptions
- This page is for direct beat sales before BeatStars.
- Post-sale delivery is still manual unless you wire your own automation.
- Exclusive licenses should remain manual until your contract and file delivery flow are finalized.
